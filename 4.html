System Overview

Data Acquisition Unit (DAU)
Microcontroller- ATMEL 89C52 is heart of the DAU. Mobile component of the system. Its function is to gather physiological information from sensors and forward to CSU for processing and verification purposes.

Central System Unit (CSU)
It contains a Bluetooth module and a PCM codec for voice data transmission. It serves as Connection Management, Data processing and recording, Access verification, System Maintenance.
